# PowerBi_Dashboard_Project
Analyzed IPL data (matches, teams, players) using Power BI with sources from Kaggle, and official datasets; performed data cleaning, transformation, and modeling.
Designed interactive dashboards with bar/pie charts, maps, filters, and slicers to visualize team stats, player performance, match outcomes, and venue trends.
